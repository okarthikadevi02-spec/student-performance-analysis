# Student Performance Analysis and Prediction

## Objective
To analyze student behavior and predict their performance (Low, Medium, High) using machine learning techniques.

## Dataset Description
The dataset contains student-related information such as:
- Gender, Nationality, Place of Birth
- Stage and Grade Level
- Number of raised hands
- Visited resources
- Announcements viewed
- Discussion participation
- Parent survey responses
- Absence days

## What I Did
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Data visualization using graphs
- Converted categorical data into numerical form
- Built machine learning models

## Visualizations
- Count plots (Class distribution, Gender vs Class)
- Box plots (Raised hands vs Class, Resources vs Class)
- Heatmap (Feature correlation)

## Machine Learning Models Used
- Logistic Regression
- Decision Tree
- Random Forest

## Evaluation
- Accuracy score comparison
- Confusion Matrix
- Classification Report

## Tools and Technologies
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

## Result
Random Forest model performed best among all models.

## Conclusion
This project demonstrates how student performance can be predicted using data analysis and machine learning techniques.

- Improve model accuracy using advanced algorithms
- Deploy the model using a web application
- Use more real-world datasets for better performance
